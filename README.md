# Anantara Vacation Club Knowledge Quiz Engine

## Overview

This repository contains a structured knowledge-check system based on official Anantara Vacation Club (AVC) materials, internal program rules, and operational guidelines.

The purpose of this project is to help users accurately verify, retain, and audit their understanding of Anantara Vacation Club programs through clear, scenario-driven Yes/No questions.

The content is designed to reflect real operational logic, QA review standards, and audit-safe interpretations of AVC policies.
